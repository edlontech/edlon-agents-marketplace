# Edlon Tech Plugins

A Claude Code plugin marketplace maintained by [Edlon Tech](https://github.com/edlontech).

> AI Toolings for your day-to-day suffering.

## Adding the marketplace

In Claude Code, register this repository as a marketplace:

```
/plugin marketplace add edlontech/edlon-agents-marketplace
```

Then browse and install plugins from it:

```
/plugin
```

## Available plugins

| Plugin | Description |
| --- | --- |
| [`gingko-memory`](https://github.com/edlontech/gingko-mem) | Persistent graph-based memory for Claude Code via the Gingko MCP server. |
